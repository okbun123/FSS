# Football Career Simulation MVP

Browser-playable football player career simulation game built as a static Vite, React, and TypeScript app for GitHub Pages.

## Local Setup

```sh
npm install
npm run dev
```

## Validation

```sh
npm run build
npm run test
```

## Scripts

- `npm run dev`: start the Vite development server.
- `npm run build`: type-check and build the static app.
- `npm run preview`: preview the production build locally.
- `npm run test`: run Vitest once.

## Current Scope

Milestone 1 scaffolds the app shell only:

- Start screen
- Player creation screen
- Career dashboard placeholder
- Local player profile persistence
- Responsive Korean UI
- Vitest setup

Full simulation systems will be added in later milestones.
